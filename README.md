## Project Title / 아웃소싱 프로젝트

- 회원 전용 사이트로, 작성한 설문 내용 기반의 영상 추천
- 유저의 실력, 관심사를 키워드로 활용
- 교육, 과학기술 분야로 카테고리 제한
- 영상 목록을 저장해 마이페이지에서 확인 가능
